# 🏏 IPL Dashboard

This project is an interactive Power BI dashboard that analyzes IPL player performance.

## 📊 Features
- Player selection using slicer
- KPI cards (Runs, Strike Rate, Sixes, Fours, Balls Faced)
- Season-wise performance visualization

## 🛠 Tools Used
- Power BI
